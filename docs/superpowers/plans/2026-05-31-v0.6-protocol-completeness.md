## Task 1: Re-auth gate helper (`pkg/authn/reauth.go`)

**Goal:** A cross-protocol single-use re-auth gate: on demand, mint a nonce + stash the demand instant in KV; on return, verify the session's `auth_time` post-dates the demand and consume the marker (single-use).

**Files:** Create `pkg/authn/reauth.go`, `pkg/authn/reauth_test.go`

**Acceptance Criteria:**
- [ ] `DemandReauth(ctx, kv kv.Store, keyPrefix string) (nonce string, err error)` — 16-byte `crypto/rand` base64url nonce; `SetEx(keyPrefix+nonce, <RFC3339Nano now>, ReauthMarkerTTL)` where `ReauthMarkerTTL = 10*time.Minute`.
- [ ] `ConsumeReauth(ctx, kv kv.Store, keyPrefix, nonce string, authTime time.Time) (bool, error)` — `Get` the marker; if missing (`kv.ErrKeyNotFound`) → `(false, nil)`; parse the stored instant `T`; `Del` the marker (single-use, before returning); return `(authTime.After(T) || authTime.Equal(T), nil)`. Other Get/Del errors propagate.
- [ ] A stale session (`authTime` before the demand) → `false`. A fresh re-login (`authTime >= T`) → `true`. Reusing a consumed nonce → `false` (marker deleted).

```go
package authn

import (
	"context"
	"crypto/rand"
	"encoding/base64"
	"errors"
	"time"

	"prohibitorum/pkg/kv"
)

// ReauthMarkerTTL bounds how long a forced-re-auth demand stays valid. The user
// must complete the login bounce + return within this window.
const ReauthMarkerTTL = 10 * time.Minute

// DemandReauth records a forced-re-authentication demand: it mints a single-use
// nonce and stamps the demand instant in KV under keyPrefix+nonce. The caller
// embeds the returned nonce in the login return_to; ConsumeReauth verifies it on
// return. keyPrefix namespaces the two protocols (e.g. "oidc:reauth:" /
// "saml:reauth:").
func DemandReauth(ctx context.Context, store kv.Store, keyPrefix string) (string, error) {
	buf := make([]byte, 16)
	if _, err := rand.Read(buf); err != nil {
		return "", err
	}
	nonce := base64.RawURLEncoding.EncodeToString(buf)
	if err := store.SetEx(ctx, keyPrefix+nonce, time.Now().UTC().Format(time.RFC3339Nano), ReauthMarkerTTL); err != nil {
		return "", err
	}
	return nonce, nil
}

// ConsumeReauth verifies (single-use) that the session's authTime post-dates the
// demand recorded under keyPrefix+nonce. It deletes the marker regardless of the
// outcome (single-use). Returns false (not an error) when the marker is missing
// or expired, or when authTime predates the demand — the caller then re-demands.
func ConsumeReauth(ctx context.Context, store kv.Store, keyPrefix, nonce string, authTime time.Time) (bool, error) {
	if nonce == "" {
		return false, nil
	}
	key := keyPrefix + nonce
	val, err := store.Get(ctx, key)
	if errors.Is(err, kv.ErrKeyNotFound) {
		return false, nil
	}
	if err != nil {
		return false, err
	}
	// Single-use: consume the marker before evaluating.
	if derr := store.Del(ctx, key); derr != nil {
		return false, derr
	}
	demandedAt, perr := time.Parse(time.RFC3339Nano, val)
	if perr != nil {
		return false, nil // corrupt marker → treat as unsatisfied; caller re-demands
	}
	return !authTime.Before(demandedAt), nil
}
```
